Perguntas frequentes

E algumas dicas

  • Qual o intervalo de tempo para atualizar as informações?

    Geralmente, iremos atualizar o site duas ou três vezes ao dia. Na seção de dados rápidos, colocamos o horário da última atualização do site.

  • Onde os dados estão armazenados?

    Colocamos todos os dados em um repositório do Github e todos os dados também estão acessíveis a todos os cidadãos. Por favor, acesse o link abaixo: repositório de dados

  • Qual framework é utilizado para visualização dos dados?

    Nós estamos utilizando o Vega-lite, que está disponível gratuitamente no link abaixo: vega-lite

  • As visualizações são acessíveis para daltônicos?

    Nós estamos adaptando as cores dos mapas e dos gráficos para qualquer pessoa com daltonismo, garantindo assim, acessibilidade a todos. Nos baseamos no estudo do David Nichols, que possui PhD em matemática, para adaptar nossas visualizações aos daltônicos. Você pode acessar o estudo no link abaixo estudo de paletas de cores para daltônicos

Um pouco sobre o projeto
